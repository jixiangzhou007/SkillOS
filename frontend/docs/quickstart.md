# 快速开始 · Personal Free

## 1. 注册

打开 `/login.html` 注册账号，自动获得 Personal Free 工作区。

## 2. 创建第一个技能

在对话区描述工作流程，例如：

> 帮我创建一个处理客户投诉的技能

多轮对话完善后，技能自动保存到「我的技能」。

## 3. 认识论确认

打开技能详情 → **认识论** Tab → 确认待审声明。

## 4. 创建团队（可选）

用户菜单 → **创建团队** → 将个人技能 **复制到公司**。

## 5. MCP（Cursor）

设置环境变量 `SKILLOS_MCP_TOKEN` 后，在 Cursor 中连接 SkillOS MCP。

## 限额

| 项目 | Personal Free |
|------|---------------|

| LLM/月 | 50 |
| BYOK | 设置 → 用量 → 自带 API Key |
