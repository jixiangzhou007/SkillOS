# 技能名称：合同审核（初版）
## Core Problem
对合同文本进行系统性审核，识别风险条款并给出修改建议，保护己方利益。

## S_route
| Intent | Action | Resource |
| :--- | :--- | :--- |
| 条款比对 | 将待审合同与**标准模板/历史版本/谈判纪要**进行逐条比对，标记差异点。 | 合同标准模板、历史合同库、谈判记录 |
| 风险识别与分级 | 对差异条款进行风险评估，按“高风险（必须改）”、“中风险（建议改）”、“低风险（仅标注）”三级分类。 | 风险分级标准表、法律法规库 |
| 修改建议与谈判策略 | 针对中、高风险条款，输出修改措辞建议，并提供谈判底线与让步策略。 | 修改措辞库、谈判话术模板 |
