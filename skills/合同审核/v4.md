# 技能名称：合同审核
## 核心问题
自动审核销售合同（软件集成类）的关键条款，包括内容价格、质保、免责、知识产权归属及对等违约金。

## S_body
1. **接收并解析合同文件**  
   - 读取用户上传或对方发来的合同文件（支持格式：PDF、Word、纯文本）。  
   - 提取合同全文文本，准备后续分析。

2. **审核内容与价格**  
   - 检查合同是否包含明确的合同标的、服务范围、交付物描述。  
   - 检查价格条款：是否写明总价、付款方式、付款节点、税率。  
   - 若价格缺失或模糊，标记为「待确认」并提示用户补充。

3. **审核质保条款**  
   - 查找质保期、质保范围、质保响应时间、质保费用承担方。  
   - 若质保期短于行业惯例（如1年）或未明确，标记为「风险项」。

4. **审核免责条款**  
   - 识别免责条款（如不可抗力、责任上限、间接损失免责）。  
   - 检查免责范围是否过于宽泛（如完全免除对方过失责任），若存在，标记为「高风险」。

5. **审核知识产权归属**  
   - 确认知识产权（包括软件代码、文档、定制开发成果）的归属方。  
   - 若归属不明确或我方需永久授权对方使用，标记为「需谈判」。

6. **审核对等违约金**  
   - 检查违约金条款是否对等（如双方违约罚则一致、比例相同）。  
   - 若我方违约金高于对方或仅单方有罚则，标记为「不对等风险」。

7. **生成审核报告**  
   - 汇总所有标记项（待确认、风险项、高风险、需谈判、不对等风险）。  
   - 输出结构化报告，包含条款原文、问题描述、建议修改方向。
