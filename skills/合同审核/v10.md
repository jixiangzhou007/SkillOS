# 合同审核

## 核心问题
快速、标准化地审核销售合同，识别关键风险点并给出修改建议。

## S_trigger
- keywords: contract, 合同审核, 审合同, 合同风险, 条款审查
- context: 用户上传了文件或提出了与合同审查相关的请求
- excludes: 用户明确表示不需要审核（如“只是看看”），或文件内容明显不是合同（如发票、名片）

## S_body
Follow these steps in order. Ask the user if anything is marked [待确认].

1.  **合同接收与识别**：当用户上传文件或提及合同审核时，检查文件名是否包含 `contract` 关键词。若包含，则启动审核流程；若不包含，询问用户是否仍要按合同审核流程处理。
2.  **合同初审（关键条款提取）**：读取合同全文，提取并列出以下关键条款内容：
    -   合同价格与支付条款
    -   质保期与质保范围
    -   责任限制与赔偿条款
    -   知识产权归属条款
    -   违约金与违约责任条款
3.  **风险分析与标注**：针对上述每个关键条款，判断是否存在对甲方（我方）不利的潜在风险。例如：
    -   **IF** 责任限制条款中，对甲方（我方）的赔偿上限过低（如低于合同总金额），**THEN** 标记为“高风险”，并建议争取提高上限或删除该限制。
    -   **IF** 知识产权条款约定成果归客户所有，**THEN** 标记为“高风险”，并建议修改为双方共有或我方保留背景知识产权。
    -   **IF** 违约金比例过高（如超过合同总金额的30%），**THEN** 标记为“中风险”，并建议调整至合理范围。
4.  **生成审核报告**：汇总以上分析，生成一份结构化的审核报告，包含：
    -   合同基本信息（文件名、日期）
    -   关键条款摘要
    -   风险点列表（含风险等级：高/中/低）
    -   针对每个风险点的具体修改建议
    -   [待确认] 是否需要在报告中引用相关法条（如《民法典》）。

## S_route
| 用户意图/条件 | 执行动作 | 备注 |
| :--- | :--- | :--- |
| 用户上传文件，文件名包含 `contract` | 启动完整合同审核流程（S_body 步骤 1-4） | 自动触发 |
| 用户上传文件，文件名不包含 `contract` | 询问用户是否按合同审核流程处理 | 避免误触发 |
| 用户仅提问“帮我看看这个合同的风险” | 启动完整合同审核流程（S_body 步骤 1-4） | 关键词触发 |
| 用户要求“只检查价格和违约金条款” | 仅执行 S_body 步骤 2 和 3 中关于价格和违约金的部分，并生成报告 | 分支处理，缩小范围 |

## S_params
```markdown
## S_params
- file_path: string, 无默认值, 待审核合同文件的路径或上传的文件对象
- risk_focus: string, 默认值 "all", 指定审核重点，可选值 ["all", "price", "warranty", "liability", "ip", "penalty"]
- output_format: string, 默认值 "report", 输出格式，可选值 ["report", "summary"]
- risk_thresholds: dict, 默认值 {"liability_cap_ratio": 1.0, "penalty_ratio": 0.3, "ip_ownership": "client"}, 用于定义风险判断的阈值参数，例如：
  - liability_cap_ratio: float, 责任赔偿上限占合同总金额的比例，低于此值标记为高风险
  - penalty_ratio: float, 违约金比例上限，超过此值标记为中风险
  - ip_ownership: string, 知识产权归属方，可选值 ["client", "both", "ours"]，用于判断风险
- contract_info_extraction_rules: list, 默认值 [{"field": "price", "keywords": ["价格", "金额", "总价"]}, {"field": "warranty", "keywords": ["质保", "保修", "保证期"]}, {"field": "liability", "keywords": ["责任限制", "赔偿上限", "免责"]}, {"field": "ip", "keywords": ["知识产权", "版权", "专利"]}, {"field": "penalty", "keywords": ["违约金", "罚金", "滞纳金"]}], 用于从合同中提取关键条款的关键词规则列表，每个元素包含字段名和对应的关键词列表
- report_customization: dict, 默认值 {"include_law_references": false, "risk_level_colors": {"high": "red", "medium": "yellow", "low": "green"}}, 报告定制参数，例如是否引用法条、风险等级颜色标识等
```
