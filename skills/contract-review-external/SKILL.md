# 合同审核
## S_body
1. [门禁] 文件可读性检查：文件内容必须可正常读取（非扫描图片、非乱码、非加密） → 不满足则中止
2. [门禁] 合同类型识别：必须为销售合同（依据文件名、标题、首段判断） → 非销售合同则中止
3. [动作] 提取价格、保修、责任限制、知识产权、违约金五类条款原文，并逐项执行以下审核结构：
   - 原文引用（精确摘录条款原文）
   - 风险等级（高/中/低）
   - 风险描述（说明该条款可能带来的法律或商业风险）
   - 修改建议（给出具体修改方向或替代措辞）
4. [动作] 扫描合同全文，标记关键条款（金额、期限、违约、管辖）
5. [门禁] 合同主体信息必须完整 → 不完整则中止
6. [动作] 检查违约金比例是否超过合同总金额30%
7. [动作] 检查管辖法院是否为甲方所在地
8. [动作] 检查是否存在单方免责的格式条款
9. [动作] 汇总风险项，按高/中/低分级
10. [动作] 生成审核报告
