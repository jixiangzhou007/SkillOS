# 技能名称：销售合同审核（初版）
## Core Problem
对销售合同文本进行系统性审核，重点识别价格、付款、产品描述等核心条款的风险，保护己方利益。

## S_route
| Intent | Action | Resource |
| :--- | :--- | :--- |
| 条款比对与核心利益审查 | 将待审合同与**标准模板/历史版本/谈判纪要**进行逐条比对，**优先审查“产品/服务描述”（含规格、数量、质保期）和“价格/付款条款”（含单价、总价、付款比例、节点、违约金）**，标记所有差异点。 | 合同标准模板、历史合同库、谈判记录、[待补充：行业标准条款库] |
| 风险识别与分级 | 对差异条款进行风险评估，按“高风险（必须改）”、“中风险（建议改）”、“低风险（仅标注）”三级分类。**其中，付款比例、质保期、核心交付标准等直接影响利润或法律责任的差异，默认至少为中风险。** | 风险分级标准表、法律法规库、[待补充：公司内部风险偏好] |
| 修改建议与谈判策略 | 针对中、高风险条款，输出修改措辞建议，并提供谈判底线与让步策略。 | 修改措辞库、谈判话术模板 |
