# 数据分析周报流程。收集各部门数据

## 核心问题
每周自动完成跨部门数据收集、清洗、分析、可视化，并输出含亮点/风险/建议的PPT周报，替代人工重复操作。

## When to use
- keywords: 周报，数据分析周报，出周报，周报流程，每周周报，数据周报
- context: 每周五下午需要自动生成跨部门数据分析周报，或用户临时要求生成周报时
- excludes: 用户仅询问数据查询（如「查一下上周销售额」）、非周报性质的日常数据分析、用户明确说「不要周报」

## Instructions
Follow these steps in order. Ask the user if anything is marked [待确认].

1. **检查触发条件**
   - 若当前为周五且时间≥14:00，则启动流程
   - 若数据拉取失败，自动重试3次，每次间隔30秒
   - 若重试3次仍失败，发送预警邮件至[待确认：收件人邮箱]
   - 若数据量骤降超过50%（相比上周同期），发送预警邮件至[待确认：收件人邮箱]

2. **收集数据**
   - 从MySQL抽取各部门业务表：销售、市场、运营、产品
   - 从Excel导入手工报表：财务、人力（路径：[待确认：Excel文件路径]）
   - 合并所有数据至统一DataFrame，按部门+日期作为联合主键
   - 若某部门数据缺失，跳过该部门并记录日志

3. **清洗数据**
   - 去重：按主键（订单ID、用户ID等）删除重复行，保留第一条
   - 补空值：数值型字段用中位数填充，类别型字段用众数填充，时间型字段用前值填充
   - 格式统一：日期转为datetime，金额转为float，字符串去首尾空格
   - 异常检测：计算各数值字段的均值和标准差，标记超过均值±3倍标准差的数据点为异常（不删除，仅标记）
   - 若某字段空值比例超过50%，[待确认：是否直接删除该字段]

4. **统计分析**
   - 计算描述性统计：均值、中位数、标准差、最大值、最小值
   - 对比分析：计算本周 vs 上周的环比增长率（公式：(本周值-上周值)/上周值×100%）
   - 按部门/产品线/渠道[待确认：具体拆分维度]拆分计算
   - 关键指标：[待确认：具体指标列表，如销售额、用户数、转化率、客单价]

5. **可视化**
   - 趋势图：折线图展示关键指标的周变化（横轴=周，纵轴=指标值）
   - 对比图：柱状图展示各部门/产品线本周表现
   - 图表配色：[待确认：是否使用固定配色方案]
   - 输出格式：PNG图片，保存至[待确认：输出路径]

6. **生成结论**
   - 结论分三部分：亮点、风险、建议
   - 每部分不超过3条，每条一句话
   - 亮点：环比增长>10%的指标或部门
   - 风险：环比下降>10%的指标或部门，或异常标记较多的字段
   - 建议：基于亮点和风险给出可操作建议
   - 若所有指标环比变化均在±5%以内，结论写「本周整体平稳，无显著波动」

7. **输出PPT**
   - 使用[待确认：PPT模板路径]生成PPT
   - 每页结构：标题+图表+结论文字
   - 保存至[待确认：输出路径]，文件名格式：周报_YYYYMMDD.pptx
   - [待确认：是否需生成演讲备注]

## Decision routes
| 用户意图/条件 | 执行动作 | 备注 |
|------------|---------|------|
| 用户说「做周报」「出周报」「周报流程」 | 立即执行完整流程（忽略时间检查） | 手动触发 |
| 当前为周五14:00且无用户指令 | 自动执行完整流程 | 定时触发 |
| 数据拉取失败 | 重试3次，每次间隔30秒；3次均失败则发预警邮件 | 异常处理 |
| 数据量骤降超过50% | 发送预警邮件，继续执行后续步骤 | 异常处理 |
| 用户说「只看XX部门周报」 | 仅收集该部门数据，执行完整流程 | 单部门模式 |
| 用户说「不要PPT，只要数据」 | 执行步骤1-5，输出Excel文件，跳过PPT生成 | 输出格式切换 |

## Inputs
- mysql_host: string，默认值"localhost"，MySQL数据库地址
- mysql_port: integer，默认值3306，MySQL端口
- excel_path: string，默认值"./data/manual_reports/"，手工Excel报表所在目录
- output_path: string，默认值"./output/"，PPT和图表输出目录
- ppt_template: string，默认值"./templates/weekly_report_template.pptx"，PPT模板路径
- fill_strategy: string，默认值"median"，空值填充策略（median/mode/ffill）
- anomaly_threshold: float，默认值3.0，异常检测标准差倍数
- alert_email: string，默认值""，预警邮件收件人（空则不发送）
