# 速查表: 数据产权登记制度与三权分置解读

```markdown
# 数据产权登记速查表

## 1. 核心公式 / 规则

> **三权分置 = 持有权 + 使用权 + 经营权**  
> 同一数据可被多个主体同时享有同一权利，互不排斥。

> **约定优先 → 法定补充**  
> 市场主体先通过合同约定权利归属；约定不明时，法律提供默认规则。

> **统一登记凭证 = 交易 + 资产确认 + 融资抵押的核心权属证明**

## 2. 决策流程（if-then 分支）

- **如果** 你想委托中介销售数据 → **授予经营权**（不授予持有权、使用权）
- **如果** 你想让第三方（如药企）加工分析数据 → **授予使用权**（不授予持有权、经营权）
- **如果** 出现权属争议 → **优先问“数据怎么用”**，而非“数据是谁的”
- **如果** 中介达成交易意向 → **先复核需求方可信度**，再实际提供数据

## 3. 常见陷阱

- ❌ 把数据产权等同于“所有权” → 应聚焦“三权分置”的拆分使用
- ❌ 忽略“低成本复制”特性 → 同一数据可被多主体同时使用，不必独占
- ❌ 在权属不清时纠缠归属 → 应转向使用方式、用途和收益分配
- ❌ 未做登记凭证 → 无法获得权威权属证明，阻碍交易与融资

## 4. 关键数字 / 阈值

- **三权**：持有权、使用权、经营权（三个独立维度）
- **一个凭证**：数据产权登记凭证（市场交易、资产确认、融资抵押的核心证明）
- **一个乘数效应**：同一数据被多个主体重复使用，创造多样价值增量

## 5. 一句话总结

> **数据产权登记 + 三权分置 = 破解“权属不清、流通不畅”的核心钥匙，是数据从资源变资产的根本前提。**
```
