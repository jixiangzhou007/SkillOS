# 技能名称：淘宝小店数码配件退款处理流程
## Core Problem
处理淘宝小店（日单200单，主营数码配件）的退款申请，确保流程合规、减少纠纷。

## S_route
| Intent | Action | Resource |
| :--- | :--- | :--- |
| 发起退款 | 1. 核对订单信息：确认订单号、用户、商品、金额与退款申请一致。<br>   - **【新增】金额核对要点**：以订单系统“实付金额”为准，区分“买家转账金额”与“优惠券/平台补贴抵扣金额”。<br>   - **【新增】付款渠道核对**：确认支付宝/微信的支付单号与订单号关联，防止套用他人付款截图。<br>2. 按订单状态分流：<br>   - **未发货**：直接同意全额退款。<br>   - **已发货/未签收**：联系快递拦截，拦截成功则全额退款。<br>   - **已发货/已签收**：进入“人工审核”流程。<br>3. 边界情况处理：<br>   - 超过售后期（如购买超30天）：拒绝申请。<br>   - **【新增】已签收-质量问题/描述不符**：要求买家提供清晰照片/视频凭证，核实后处理。<br>   - **【新增】已签收-七天无理由**：确认商品不影响二次销售，同意退货，运费按店铺规则处理。 | 订单系统、聊天记录、物流信息 |
