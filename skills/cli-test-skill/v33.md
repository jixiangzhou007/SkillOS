```yaml
## S_body
1.  **合同审核流程**
    *   **步骤1：初步审查**
        *   核对合同主体信息（名称、地址、法定代表人等）是否与营业执照一致。
        *   确认合同标的、数量、质量、价款等核心商业条款是否明确、无歧义。
    *   **步骤2：关键条款审查（分支处理）**
        *   **2a. 违约责任条款**：检查违约情形、违约金比例（是否过高或过低）、赔偿范围（是否包含间接损失）。
        *   **2b. 争议解决条款**：确认管辖法院或仲裁机构是否明确、对己方是否有利（如约定在己方所在地）。
        *   **2c. 知识产权条款**：若涉及技术或内容，审查知识产权归属、使用范围、侵权责任分担。
        *   **2d. 保密与数据条款**：检查保密期限、保密范围、数据处理的合规性（如GDPR或《个人信息保护法》要求）。
    *   **步骤3：最终复核与输出**
        *   汇总上述审查发现的问题，形成修改建议清单。
        *   输出审核结论（通过/有条件通过/退回修改）。
```
