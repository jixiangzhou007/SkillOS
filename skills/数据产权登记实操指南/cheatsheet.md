# 速查表: 数据产权登记实操指南

```markdown
# 数据产权登记速查表（30秒版）

## 1. 核心规则

> **登记前提**：数据来源必须合法合规，禁止登记涉及国家安全、商业秘密、个人隐私等非法数据。

**分级管理模型**：
- **国家**：制定制度、建设平台、遴选机构、考核
- **省级**：推荐机构、日常指导与监督

**登记机构准入条件**（硬性门槛）：
- 实缴资本、从业经验年限、安全等级达标
- 退出时需提前报备方案，确保凭证持续有效

## 2. 决策流程（if-then 分支）

```text
if 数据来源不合法 or 涉及禁止情形:
    → 拒绝登记
else:
    → 进入七步程序：申请 → 受理 → 审查 → 公示 → 异议处理 → 信息存证 → 核发凭证

if 申请材料不全:
    → 登记机构在3个工作日内一次性告知补正内容
    → 避免反复退回

if 审查通过:
    → 设置公示期，接受社会监督
    → 任何单位或个人可提出异议 → 登记机构调查处理

if 数据产权发生变动:
    → 选择对应登记类型：
        - 初次登记（首次确权）
        - 转让登记（产权转移）
        - 变更登记（非根本变化）
        - 续期登记（延续效力）
        - 注销登记（清除失效记录）
```

## 3. 常见陷阱（避免）

- ❌ **数据来源不清**：未核实合规性就申请登记 → 被驳回
- ❌ **材料不全未一次性补正**：机构未在3日内告知 → 可投诉
- ❌ **忽视公示异议期**：未关注公示，错过提出异议机会
- ❌ **混淆登记类型**：如将“转让”当作“变更”申请 → 程序错误
- ❌ **退出机构未提前报备**：导致已发凭证失效 → 机构责任

## 4. 关键数字/阈值

| 项目 | 数值/要求 |
|------|-----------|
| 补正告知时限 | 3个工作日（一次性告知） |
| 登记机构准入 | 实缴资本、经验年限、安全等级（具体数值未给出，需参考细则） |
| 公示期 | 未明确天数（需关注具体办法） |
| 登记类型 | 5种（初次、转让、变更、续期、注销） |

## 5. 一句话总结

> **数据产权登记 = 合法合规来源 + 七步闭环程序 + 五种类型覆盖全生命周期 + 凭证可复用（交易、入表、融资、纠纷）。**
