# Skill Name: Customer Refund Processing
## Core Problem
Process a customer refund by verifying the order, checking policy, calculating the amount, and issuing payment.
## S_route
| Intent | Action | Resource |
| Initiate refund | Verify order ID and purchase date | Order management system |
| Initiate refund | Check refund policy (30-day window) | Refund policy database |
| Initiate refund | Calculate refund amount (product + tax - shipping) | Pricing/order system |
| Initiate refund | Process refund through payment gateway | Payment gateway API |
| Initiate refund | Send confirmation email to customer | Email notification system |
## S_body
1. Verify order ID and purchase date
2. Check refund policy (30-day window)
3. Calculate refund amount (product + tax - shipping)
4. Process refund through payment gateway
5. Send confirmation email to customer
## S_trigger
- keywords: refund, return, customer refund, process refund, issue refund
## S_params
- order_id: The unique identifier for the customer's purchase
- purchase_date: The date the order was placed
- product_price: The cost of the product(s) being refunded
- tax_amount: The tax charged on the order
- shipping_cost: The shipping fee paid by the customer
