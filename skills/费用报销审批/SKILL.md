# 技能名称：费用报销审批
## 核心问题
规范创业公司员工在飞书提交的费用报销审批流程，自动生成付款台账并同步财务系统。

## S_body
1. 接收飞书「费用报销」模板提交的申请，检查附件是否包含发票 PDF 和付款截图；若缺少任一附件，直接驳回并提示「请补充发票 PDF 和付款截图」。
2. 核对申请人部门、费用类型（差旅/办公/招待）、金额和发票抬头是否为公司全称；若发票抬头非公司全称，驳回并提示「发票抬头需为公司全称」。
3. 根据费用类型执行分支：
   - 若费用类型为「差旅」，检查是否附带行程单；若缺少行程单，驳回并提示「差旅费必须附带行程单」。
   - 若费用类型为「招待」，检查是否填写客户名称；若未填写，驳回并提示「招待费必须写清楚客户名称」。
4. 根据金额执行审批路由：
   - 若金额 < 500元，自动转部门负责人审批。
   - 若金额 >= 500元且 <= 5000元，自动转CFO审批。
   - 若金额 > 5000元，自动转CFO审批后加签CEO审批。
5. 检查发票号是否重复：若同一发票号已存在报销记录，系统自动拦截并通知财务人员；若未重复，继续流程。
6. 若审批通过，自动生成付款台账Excel，同步到用友云，并在飞书群「财务通知」发送已打款提醒（含报销单号和金额）。
7. 若审批被驳回，记录驳回原因并通知申请人。
8. 若为紧急采购且走「预付款」流程，允许先提交无发票申请，事后补发票；补发票时触发重新核对流程。

- 金额: number, 无默认值，单位元

- 客户名称: string, 可选，仅招待费必填
- 付款截图附件: file, 必填
