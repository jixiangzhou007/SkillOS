# 速查表: 数据产权登记凭证的效力与边界

```markdown
# 数据产权登记凭证速查表

## 1. 核心公式/规则

> **凭证效力 = 审查 + 公示 + 存证（闭环）**

- **审查**：登记机构对数据准确性、合规性、产权明确性进行合理审慎审查（非全面背书）。
- **公示**：信息公示不少于固定期限，保留异议通道；异议成立则不予发证或先予纠正。
- **存证**：所有登记事项同步存证于统一平台，凭证与平台信息不一致时，**以平台为准**。

## 2. 决策流程（if-then 分支）

```
if 数据来源违法、侵权或隐私违规:
    → 登记凭证可能被穿透追责（凭证非免责金牌）

if 凭证信息与统一存证平台不一致:
    → 以平台信息为准

if 凭证超过有效期（通常≤5年）:
    → 需办理续期、变更或注销

if 公示期内收到异议且成立:
    → 不予发证或先予纠正

if 需要司法/仲裁举证:
    → 凭证可作为初步证据，帮助完成初步举证责任
```

## 3. 常见陷阱

- ❌ **凭证 = 免责金牌**：凭证不代表国家为所有用途背书，来源违法、侵权、隐私违规仍可能被追责。
- ❌ **凭证 = 一发永逸**：凭证有有效期（≤5年），需周期性复核、续期或变更。
- ❌ **凭证 = 唯一权威**：凭证与统一存证平台信息不一致时，以平台为准。
- ❌ **凭证 = 全面背书**：登记机构仅做有限审查，不保证数据绝对无误或无争议。

## 4. 关键数字/阈值

| 项目 | 数值/要求 |
|------|-----------|
| 公示期限 | 不少于固定期限（文档未明确具体天数，但强调“不少于”） |
| 凭证有效期 | 不超过5年 |
| 异议处理 | 异议成立 → 不予发证或先予纠正 |
| 存证平台 | 统一平台，凭证与平台不一致时以平台为准 |

## 5. 一句话总结

> **数据产权登记凭证的权威性来自“审查-公示-存证”闭环，而非凭证本身；它是有条件的合规证明和初步证据，但绝非免责金牌或永久通行证。**
```
