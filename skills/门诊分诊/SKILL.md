# 门诊分诊

## 核心问题
将挂号后的门诊病人，通过标准化流程快速完成生命体征采集、病情评估与分级，并准确分配至对应科室，确保急危重症患者优先处理。

## S_trigger
- keywords: 门诊分诊, 分诊, 分级, 安排科室, 急诊分级, 孕妇优先, 测体温, 测血压, 主诉, 病史
- context: 用户描述或询问门诊病人挂号后的分诊流程、病情分级标准、科室分配规则，或需要系统自动执行分诊决策时
- excludes: 住院病人分诊、急诊抢救室分诊、手术室分诊、院前急救分诊

## S_body
Follow these steps in order. Ask the user if anything is marked [待确认].

1. 病人挂号后，测量体温和血压。
   - 若体温 ≥ 39°C 或收缩压 ≥ 180mmHg 或舒张压 ≥ 110mmHg，直接转入急诊流程（无需继续分诊）。
   - 若体温 ≥ 37.3°C 且 < 39°C，标记为发热病人，后续安排至发热门诊或感染科。
   - 若血压收缩压 ≥ 140mmHg 或舒张压 ≥ 90mmHg，标记为高血压病人，后续安排至心血管内科。
2. 询问病人主诉及病史。
   - 若病人无法清晰表述（如老年人、儿童、语言障碍者），优先询问陪同家属或监护人；若无陪同人员，查阅挂号时登记的历史病历或电子健康档案。
   - 若仍无法获取有效信息，标记为「信息不全」，安排至全科门诊或导诊台人工处理。
3. 根据收集到的信息进行病情分级：
   - 一级（急危重症）：生命体征异常（如高烧、高血压危象、呼吸困难、意识模糊）→ 直接转入急诊。
   - 二级（紧急）：症状较重但生命体征稳定（如剧烈疼痛、持续呕吐、高热不退）→ 优先安排至对应专科，并通知护士站提前准备。
   - 三级（普通）：常见病、慢性病复诊、轻微不适 → 按正常顺序安排至对应科室。
   - 四级（非紧急）：健康咨询、体检、开药 → 安排至全科门诊或健康管理中心。
4. 根据分级结果安排科室：
   - 一级 → 急诊科。
   - 二级 → 对应专科（如心内科、呼吸科、消化科等），优先插队。
   - 三级 → 对应专科，正常排队。
   - 四级 → 全科门诊或健康管理中心。
5. 特殊人群优先处理：
   - 若病人为孕妇，无论分级如何，均优先安排就诊（在同等分级内插队至最前）。
   - 若孕妇同时满足一级或二级条件，按最高优先级处理。

- 舒张压: int，默认值 80，范围 40-160，单位 mmHg
