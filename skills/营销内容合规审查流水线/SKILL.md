# 营销内容合规审查流水线

## 核心问题
确保市场部提交的营销文案在发布前完成广告法合规审查、虚假宣传审查及特殊行业（医疗/金融/教育）外部复审，通过后归档，不通过则标注违规条款退回。

## When to use
- keywords: 营销文案审查, 广告法合规, 虚假宣传审核, 营销内容审批, 合规审查, 文案审核, 广告审核
- context: 市场部在OA系统提交营销文案后，需要法务部门进行合规审查的场景。适用于企业营销内容发布前的标准化审核流程。
- excludes: 不适用于非营销内容的审核（如内部通知、技术文档）；不适用于已经发布后的内容追溯审查；不适用于合同审查、知识产权审查等其他法务工作。

## Instructions
Follow these steps in order. Ask the user if anything is marked [待确认].

1.  **收件与预检**
    -   **输入**：市场部通过OA系统提交的营销文案全文、产品说明、投放渠道。
    -   **动作**：
        -   记录提交时间、提交人、产品/服务名称。
        -   **关键判断**：识别文案中是否包含“医疗”、“金融”、“教育”、“治疗”、“投资回报”、“保本”等敏感关键词或暗示此类概念的表述。
        -   **预检结果**：
            -   若未触发敏感词，进入“法务初审”环节。
            -   若触发敏感词（医疗/金融/教育），标记为“高风险-需外部复审”，直接进入“外部合规顾问复审”环节。
    -   **时效**：[待确认：1个工作日内完成预检]

2.  **法务初审（双轨并行）**
    -   **第一步：广告法合规审查**
        -   **检查项**：是否使用“第一”、“最好”、“国家级”等绝对化用语（出现任意一个直接退回）；是否使用或变相使用国旗、国歌、军旗等；是否损害国家尊严或利益；是否含有淫秽、迷信、暴力等内容；是否违反《广告法》其他禁止性规定。
        -   **输出**：若违反，记录违规条款，进入“不通过-标注违规条款”流程。
    -   **第二步：虚假宣传审查**
        -   **检查项**：文案中的性能、功能、产地、用途、质量、成分、价格、生产者、有效期限、允诺等是否真实、准确；是否使用虚构、伪造或者无法验证的科研成果、统计资料、调查结果、文摘、引用语等信息；是否捏造、散布虚假事实，损害竞争对手的商业信誉、商品声誉。
        -   **输出**：若存在虚假宣传嫌疑，标记为“疑似虚假宣传”，要求市场部提供证明材料（如检测报告、临床数据、专利证书等）。
    -   **分支逻辑**：
        -   若“广告法合规”与“虚假宣传”均通过，进入“终审归档”环节。
        -   若“广告法”不通过（含红线词），直接“不通过-标注违规条款”退回。
        -   若“虚假宣传”存疑，进入“待补充证据”状态，等待市场部反馈。

3.  **外部合规顾问复审（条件触发）**
    -   **触发条件**：预检阶段识别出涉及医疗、金融、教育产品的营销文案。
    -   **审查内容**：由外部合规顾问对文案全文、产品说明、投放渠道进行专项合规审查，重点检查行业监管法规（如《药品管理法》、《医疗器械监督管理条例》、《商业银行法》、《广告法》特殊行业条款等）的符合性。
    -   **输出**：
        -   审查通过：出具《外部合规审查意见书》，进入“终审归档”环节。
        -   审查不通过：出具《外部合规审查意见书》并标注违规条款，进入“不通过-标注违规条款”流程。

4.  **终审归档或退回**
    -   **通过归档**：对审查通过的营销文案添加“已通过合规审查”水印，归档至内容管理系统，记录审查人、审查时间、审查结论。
    -   **不通过退回**：将标注了具体违规条款的文案退回至市场部提交人，附上审查意见，要求修改后重新提交。

## Decision routes
| 用户意图/条件 | 执行动作 | 备注 |
|------------|---------|------|
| 市场部提交营销文案，未触发敏感词 | 执行法务初审（广告法合规审查 + 虚假宣传审查） | 双轨并行 |
| 法务初审发现广告法红线词（第一/最好/国家级） | 直接退回，标注违规条款 | 无需等待虚假宣传审查结果 |
| 法务初审发现虚假宣传嫌疑 | 标记为“待补充证据”，等待市场部反馈证明材料 | 市场部需在[待确认]个工作日内反馈 |
| 预检触发敏感词（医疗/金融/教育） | 跳过法务初审，直接进入外部合规顾问复审 | 特殊行业强制外部复审 |
| 外部合规顾问复审通过 | 添加水印，归档至内容管理系统 | 记录审查信息 |
| 外部合规顾问复审不通过 | 退回并标注违规条款 | 附《外部合规审查意见书》 |
| 法务初审全部通过（无红线词、无虚假宣传） | 添加水印，归档至内容管理系统 | 无需外部复审 |

## Inputs
- marketing_content: string, 必填，市场部提交的营销文案全文（文本、图片描述、视频脚本等）
- product_description: string, 必填，产品/服务说明文档
- distribution_channel: string, 必填，计划投放的渠道（如官网、社交媒体、电视、户外广告等）
- submitter: string, 必填，提交人姓名/工号
- product_category: string, 必填，产品所属类别（普通/医疗/金融/教育/其他）
- submission_time: datetime, 自动记录，OA系统提交时间
- evidence_materials: array, 可选，虚假宣传审查时市场部补充的证明材料

## Outputs
- status: enum — 执行结果（success / pending / rejected / escalated）
- message: string — 给用户或下游系统的摘要说明
- audit_notes: string — 关键操作记录（可选）
