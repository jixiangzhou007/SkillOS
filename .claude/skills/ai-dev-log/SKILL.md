# AI 开发日志写入

按 SkillOS 项目 AGENTS.md 强制协作协议 #2 的要求，每次完成一轮代码修改后，必须在 docs/AI_DEV_LOG.md 顶部追加一条记录。

## 模板格式

```
## [YYYY-MM-DD] 标题 — Claude Code
**背景 / 触发**：[本次修改的原因]
**修改思路**：[设计思路摘要]
**修改内容**：文件变更表
**未修改 / 刻意不做**：[刻意不变更的内容及原因]
**验证**：[执行的命令及结果]
**开放问题 / 下一步**：[后续待处理事项]
```

## 执行步骤

1. 回顾当前会话中所有已修改的文件
2. 使用上面的模板撰写一条日志记录，所有 7 个字段必填
3. 若未运行验证命令，必须在「验证」字段中说明原因
4. 将新记录插入到 docs/AI_DEV_LOG.md 文件顶部（最新记录在最上面）
5. 不要在未修改任何文件时调用此技能
